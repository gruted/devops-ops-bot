# Money POC 1 Summary

- Created a CLI tool that fetches the current Bitcoin price from CoinDesk API.
- It compares the current price with a saved previous price.
- If the price changes by $100 or more, it outputs an alert message.
- Everything is self-contained and can be run repeatedly, suitable for cron scheduling.
- The proof of concept demonstrates how automated alerts can be monetized by adding real-time notification channels or premium data analytics.
- README included with usage and monetization ideas.

This serves as a baseline for more advanced monetizable automation prototypes.
