# OpenClaw Ops (Space)

## Purpose
Ongoing operations for the VPS + OpenClaw: deploys, incident response, reliability, security hardening, backups.

## Where things live
- Primary trackers: `STATE.md` and `TASKS.md`
- Runbooks/notes: `notes/`

## House rules
- Unencrypted rooms for bot compatibility; don’t paste long-lived secrets.
- Link to Bitwarden SM keys instead of pasting values.

## Suggested workflow
- Use **ops-workshop** for day-to-day.
- Post major changes in **ops-announcements**.
- Route monitoring noise and automation outputs to **ops-alerts**.
